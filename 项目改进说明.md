## 六、游戏规则

### 1. 猜数字模式（无限次数）
- 系统随机生成目标数字
- 玩家无限次猜测
- 系统提示"偏大"或"偏小"
- 记录猜测次数

### 2. 挑战模式（限定次数）
- 玩家设定最大尝试次数
- 在限定次数内猜中即为胜利
- 超过次数未猜中即为失败
- 难度评估：
  - **就这**：设定次数 > log₂(区间长度)
  - **实力**：设定次数 < log₂(区间长度)
  - **一般**：设定次数 = log₂(区间长度)

### 3. 难度计算
```javascript
// 计算最优猜测次数
function calculateLog2n(min, max) {
    const range = max - min + 1;
    return Math.ceil(Math.log2(range));
}

// 普通模式难度
if (log2n <= 5) return "简单";
if (log2n <= 10) return "一般";
if (log2n <= 15) return "困难";
if (log2n <= 20) return "地狱";
return "噩梦";
```

### Q2：WebSocket 连接失败怎么办？

**检查清单**：
1. 确认服务器地址正确
2. 检查防火墙是否开放 8080 端口
3. 查看服务器日志：`tail -f app.log`
4. 测试 HTTP 接口是否正常：`curl http://localhost:8080/game`

### Q3：数据库连接失败？

**解决步骤**：
1. 检查 MySQL 服务是否启动：`sudo systemctl status mysql`
2. 验证数据库凭证：`mysql -u root -p`
3. 检查配置文件中的数据库连接信息
4. 查看应用日志中的详细错误信息

### Q4：如何清空所有游戏数据？
