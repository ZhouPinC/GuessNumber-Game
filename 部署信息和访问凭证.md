修改密码的方法：

**方法1：通过数据库修改**
```sql
-- 1. 生成新密码的 BCrypt 哈希值
-- 在线工具：https://bcrypt-generator.com/
-- 输入您的新密码，选择 rounds=10

-- 2. 更新数据库（如果有用户表）
UPDATE user SET password = '$2a$10$您的新密码哈希值' WHERE username = 'admin';
```

**方法2：修改 Spring Security 配置**
```java
// 在 SecurityConfig.java 中修改
@Bean
public UserDetailsService userDetailsService() {
    UserDetails admin = User.builder()
        .username("admin")
        .password(passwordEncoder().encode("您的新密码"))
        .roles("ADMIN")
        .build();
    return new InMemoryUserDetailsManager(admin);
}
```

## 🔧 技术支持

### 常见问题

**Q1：页面无法访问怎么办？**
- 检查网络连接
- 确认 URL 地址正确
- 尝试刷新页面（Ctrl+F5）
- 清除浏览器缓存

**Q2：登录失败？**
- 确认用户名和密码正确（区分大小写）
- 检查是否有输入空格
- 尝试使用其他浏览器

**Q3：游戏数据不同步？**
- 检查 WebSocket 连接状态
- 查看浏览器控制台是否有错误
- 刷新页面重新连接

**Q4：后台管理页面显示空数据？**
- 这是正常的，因为还没有游戏记录
- 先玩几局游戏，数据会自动同步到后台

### 日志查看

如需查看服务器日志：
```bash
# 查看应用日志
```
